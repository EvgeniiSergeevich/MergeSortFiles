Программа для сортировки слиянием нескольких файлов.
Параметры запуска:
	-обязательные:
		"-s" - сортировка слиянием файлов со строками
		"-i" - сортировка слиянием файлов с числами
	-не обязательные:
		"-a" - сортировка по возрастанию (по умолчанию)
		"-d" - сортировка по убыванию

Так же в параметрах указываются имена файлов.
Входные файлы - все файлы с расширением .txt, которые есть в корневой папке. Кроме файла, начинающегося с "out" - он выходной.

Пример параметров для запуска: 
	-i -a out.txt in.txt in1.txt		(для целых чисел по возрастанию)
	-s out.txt in2.txt in3.txt in4.txt	(для строк по возрастанию)
	-d -s out.txt in4.txt in2.txt		(для строк по убыванию)

В результаты работы программы получается файл out.txt с отсортированными данными всех файлов, которые были указаны в параметрах запуска.

Работал в ItelliJ IDEA. JAVA 16
